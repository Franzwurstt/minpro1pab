## Deskripsi Project

Aplikasi Katalog Smartphone Samsung merupakan program yang dibuat untuk melakukan pencatatan terkait handphone merk Samsung. Di aplikasi ini pengguna bisa menambah, melihat, menghapus, dan mengubah data katalog Handphone Samsung

## Fitur Aplikasi
Create (Tambah Data) Pengguna dapat menambahkan data handphone serta menambah gambar  dengan mengisi form tambah katalog.

Read (Tampilkan Data) Aplikasi menampilkan daftar katalog handphone samsung yang sudah ditambahkan.

Update (Edit Data) Pengguna dapat mengubah data insiden seperti nama seri samsung, harga, tahun, spesifkasi dan gambar.

Delete (Hapus Data) Pengguna dapat menghapus data katalog dari daftar.

Multi Page Navigation Aplikasi memiliki beberapa halaman:

Halaman Utama untuk menampilkan Katalog yang sudah dibuat
Halaman Tambah Katalog
Halaman Edit Katalog
